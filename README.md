### ChatGPT_Web

这是一个使用flask作为后端制作的chatgptweb  
请注意把<code>main.py</code>里面<code>YOU_KEY</code>替换为你的<code>openaikey</code>  
把<code>PORT</code>更改为你需要开放的端口  
把mian.py里面相关库按照号相关库安装完成后即可运行
```
python main.py
```
启动你的脚本
